udp-network-java
================

pop quiz soln from network course written in java (2009?)

well, it was for a pop quiz. 

**usage:**

write the operator, then the operands. 

    $ javac UDPServer.java
    $ javac UDPClient.java

    $ java UDPServer
    $ java UDPClient

    $ + 3 5

**example:**

    $ java UDPClient
    Enter a sentence: + 3 5
    3 + 5 = 8
